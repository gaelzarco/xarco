RGX

Software Rasterizer

Abstract

CPU-based rendering system that converts vector graphics (like triangles) into pixel data without relying on a GPU. Handles transformations, shading, and rasterization.

Methods

Goal

  • Build a simple 3D-rendering pipeline to understand basic rasterization. Focus on limiting external dependencies.

Tools

  • Rust

Skills

  • Linear Algebra (vector/matrix operations)
  • File Parsing (.obj files)
  • 3D-Rendering (vertex transformation)
  • Rasterization (triangle filling, depth buffering)
  • Backface Culling (hidden surface removal)
  • Z-Buffering (depth management)
  • Normalization (vector normalization, coordinate mapping)
  • Interpolation (vertex attributes, shading)

Screenshots

Results

Successfully achieved the primary goal of building a minimal 3D rendering pipeline with limited external dependencies. Through implementation of each component, I developed a comprehensive understanding of how a modern graphics pipeline converts data into performant renderings.
